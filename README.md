# 毕业设计（代码+论文）
基于Arduino+Android的无线智能小车的设计与实现  
# 简述
1.硬件部分使用基于Arduino平台的ESP8266  
2.Android实现控制小车运动，Android使用kotlin编写  
3.数据查看使用web端  
4.服务器程序使用Golang编写，可轻松部署到云服务器
